# 🩺 Sistema de Saúde

Aplicação desenvolvida para auxiliar no gerenciamento de cuidados em saúde, permitindo interação com o usuário e organização de informações.

---

## 💡 Descrição

Este projeto tem como objetivo simular um sistema simples de apoio à saúde, oferecendo funcionalidades como registro de informações e exibição de dicas, com foco na usabilidade e na prática de desenvolvimento web.

---

## ⚙️ Funcionalidades

* Registro de informações (ex: medicamentos)
* Exibição de dicas de saúde
* Interação dinâmica com o usuário
* Validação de dados de entrada

---

## 🧠 Conceitos aplicados

* Manipulação do DOM
* Eventos em JavaScript
* Estruturas condicionais
* Lógica de programação

---

## 🛠️ Tecnologias utilizadas

* HTML
* CSS
* JavaScript

---

## 🚀 Como executar

1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` no navegador
3. Utilize as funcionalidades disponíveis na aplicação

---

## 📌 Objetivo do projeto

Praticar o desenvolvimento de aplicações web interativas, utilizando JavaScript para manipulação de elementos e criação de funcionalidades dinâmicas.
